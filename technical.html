How Nana Works

Frontend

Responsive web application for caregivers and children. Child profiles, pain reporting, history tracking and body-map interaction.

Body Mapping

Interactive 3D body representation allows pain location selection and visual communication.

Pain Recording

Pain intensity, area, notes and timestamps are stored and displayed in history summaries.

Backend API

REST endpoints manage children, pain reports and uploaded profile images.

Database

Children table, pain reports table and image storage buckets support application data.
